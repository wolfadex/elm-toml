# elm-toml

For parsing TOML files in Elm


Supported:
- [ ] root parser
- [x] String value
- [x] Integer value
- [x] Float value
- [x] Boolean value
- [ ] Offset Date-Time value
- [ ] Local Date-Time value
- [ ] Local Date value
- [ ] Local Time value
- [x] Array value
- [ ] Inline Table value
- [ ] tables
- [ ] array of tables
